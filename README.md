# Gravel Index

An interactive explorer for high-end race-gravel bikes — sortable, filterable, with
side-by-side compare and cited source links for every spec.

**Live site:** https://brohrich.github.io/gravel-index/

This is the public, shareable build of a personal bike-research project. It contains only
the explorer (`index.html`); fit data and other personal notes are kept private.

## Updating

`index.html` here is a copy of `bikes-explorer.html` from the private research repo (with the
private repo links trimmed). To update: copy the latest explorer over `index.html`, commit, and
push to `main` — GitHub Pages redeploys automatically.
